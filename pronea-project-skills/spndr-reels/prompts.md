# spndr Video Prompt Library

Pre-built prompts for AI video generation. All follow the spndr dark/orange aesthetic.

## Prompt Structure

```
[Camera], [Subject], [Environment], dark moody lighting with orange accent glow,
professional fintech aesthetic, cinematic depth of field, 9:16 vertical format
```

## Lifestyle Scenes

### Person Checking Phone
```
Cinematic close-up, professional person in their 30s checking smartphone with
glowing finance dashboard visible on screen, dark moody room with warm orange
accent light from screen reflecting on face, shallow depth of field, 9:16 vertical,
slow subtle camera drift, high-end commercial look
```
